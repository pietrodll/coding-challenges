# Advent of Code 2025

Solutions to [Advent of Code 2025](https://adventofcode.com/2025) challenges, implemented in Rust.

## Running Solutions

To run a specific day's solution:

```bash
cargo run <day>
```

For example, to run day 1:

```bash
cargo run 1
```

## Project Structure

- `src/days/` - Contains individual day solutions
- `data/` - Input files for each day (day1.txt, day2.txt, etc.)
- Each day module exports `run_first_part` and `run_second_part` functions
